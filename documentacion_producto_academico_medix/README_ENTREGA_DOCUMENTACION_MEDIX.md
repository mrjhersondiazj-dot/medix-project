# Entrega documental MEDIX

Fecha de generacion: 07/07/2026

Esta carpeta contiene la documentacion completa solicitada para el Producto Academico de MEDIX. No modifica el codigo del sistema; solo organiza el sustento academico, matrices, graficos y anexos.

## Archivos principales

- `Informe_Producto_Academico_MEDIX.docx`: informe editable en Word con todas las secciones solicitadas.
- `Informe_Producto_Academico_MEDIX.pdf`: version PDF del informe para entrega o revision rapida.
- `Matrices_y_Cronograma_MEDIX.xlsx`: matriz de requerimientos, cronograma, pruebas, riesgos y trazabilidad.
- `graficos/`: organigrama, arquitectura, flujo de atencion, casos de uso, resumen de requerimientos y cronograma.
